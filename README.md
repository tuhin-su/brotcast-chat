# Compile Own

## Linux or Distro
```bash
  git clone https://github.com/tuhin-su/brotcast-chat.git
  cd brotcast-chat
  pip install -r requirements.txt
  python3 main.py
```
## Windows or Distro
```bash
  git clone https://github.com/tuhin-su/brotcast-chat.git
  cd brotcast-chat
  py -m pip install -r requirements.txt
  py main.py
```
